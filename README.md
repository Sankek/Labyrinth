# Labyrinth (3D)
**3D implementation in Python nearly from scratch**

## System Requirements
    
- Python 3
- Tkinter

The game was tested on:

    Microsoft Windows Version 10.0.18362.535
    Python 3.7.4
    Tkinter 8.6
    
**(!)** In Linux there may be a problem with mouse rotations
    
 
    
## How to Play

### Moving

- **W**: forward
- **S**: back
- **A**: left
- **D**: right
- **Mouse and arrows**: look around
